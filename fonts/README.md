# Fonts Directory

## Required Font File

Place your **Slackey-Regular.ttf** font file in this directory.

### File Structure
```
fonts/
└── Slackey-Regular.ttf  ← Place your font file here
```

## Where to Get Slackey Font

If you don't have the font file yet:

1. **Google Fonts:** Visit [fonts.google.com/specimen/Slackey](https://fonts.google.com/specimen/Slackey)
2. Click "Download family"
3. Extract the ZIP file
4. Copy `Slackey-Regular.ttf` to this directory

## Usage

The Slackey font is used for:
- The "DaddyFreud" logo on the landing page
- Any other heading elements styled with `.hero-title`

## Deployment Note

When deploying to Cloudflare Pages, make sure to include this entire `fonts/` directory with the TTF file inside.
